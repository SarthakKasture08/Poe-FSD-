# Student Registration (PHP Session)

## Objective
Create a simple student registration using PHP sessions.

## Files
- index.php
- register.php
- dashboard.php
- logout.php

## How to Run
1. Install XAMPP
2. Put folder in htdocs
3. Start Apache
4. Open:
   http://localhost/12.student-registration/index.php

## Working
- Form takes input
- Data stored in session
- Display on dashboard
- Logout clears session

## Viva Points
- session_start() starts session
- $_SESSION stores data
- session_destroy() removes session

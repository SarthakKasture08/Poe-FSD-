# Question 23 - React Calculator

## Objective
To implement a simple calculator using React JS.

## Technologies Used

React JS:
- JavaScript library for UI

JavaScript:
- Handles operations

## File Used

App.js:
- Contains calculator logic

## How to Run

1. Create React app
2. Replace App.js
3. Run using npm start

## Implementation Logic

1. Two input fields take numbers
2. useState stores values
3. Buttons perform operations
4. Result displayed dynamically

## Output

- Performs addition, subtraction, multiplication, division

# AJAX GitHub API

## Objective
Use AJAX (XMLHttpRequest) to fetch data from GitHub API.

## File
- index.html

## How to Run
1. Open index.html in browser
2. Enter GitHub username
3. Click search

## Output
- Displays username
- Displays number of repositories
- Shows error if user not found

## Working
- XMLHttpRequest sends request
- GitHub API returns data
- JSON parsed and displayed

## Viva Points
- AJAX is used to fetch data without refreshing page
- XMLHttpRequest is used for API calls
- JSON is parsed using JSON.parse()

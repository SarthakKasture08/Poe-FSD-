# Question 24 - Alumni Registration Form

## Objective
To design a registration form for alumni with validation.

## Technologies Used

HTML:
- Form creation

JavaScript:
- Form validation

## File Used

index.html:
- Contains form and validation logic

## How to Run

1. Open index.html in browser
2. Fill form
3. Submit

## Implementation Logic

1. User enters data
2. validateForm() runs on submit
3. Checks:
   - Empty fields
   - Mobile number length
   - Age condition
4. If invalid → alert shown
5. If valid → form submits

## Output

- Shows alert for invalid data
- Accepts correct data

## Core Concepts

Form:
- Input collection

Validation:
- Ensures correct data

JavaScript:
- Controls logic
